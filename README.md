Política de Privacidade e Tratamento de Dados Pessoais
Este aplicativo respeita a sua privacidade e está comprometido com a proteção dos seus dados pessoais, em conformidade com a Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/2018 - LGPD).

1. Dados Coletados
Durante o uso do aplicativo, coletamos e armazenamos os seguintes dados pessoais:
Dados de identificação: nome completo, data de nascimento, foto de perfil, e-mail e senha de acesso.
Dados de treino: registros dos dias, horários, grupos musculares treinados e histórico de participação em aulas coletivas.

2. Finalidade do Tratamento
Os dados pessoais são coletados para as seguintes finalidades:

Realizar o cadastro e autenticação do usuário na plataforma;

Permitir o registro e acompanhamento de treinos e atividades físicas;

Otimizar os serviços da academia com base no histórico de uso dos alunos;

Realizar análises estatísticas e melhorias no serviço (dados anonimizados, sempre que possível).

3. Permanência dos Dados
Em caso de desligamento do aluno do sistema, os dados de identificação (como nome, e-mail, foto e senha) serão eliminados de nossa base de dados, salvo quando houver necessidade legal de retenção.
Entretanto, os dados relacionados aos treinos e à participação em aulas coletivas poderão ser mantidos exclusivamente para fins de pesquisa interna, estatísticas e aprimoramento dos serviços prestados pela academia. Esses dados serão tratados de forma anonimizada, ou seja, sem possibilidade de identificação direta ou indireta do titular, conforme previsto pela LGPD.

4. Direitos do Titular dos Dados
O usuário tem o direito de, a qualquer momento e mediante requisição:

Confirmar a existência do tratamento de seus dados;

Acessar os dados tratados;

Corrigir dados incompletos, inexatos ou desatualizados;

Solicitar a anonimização, bloqueio ou eliminação de dados desnecessários ou tratados em desconformidade com a LGPD;

Revogar o consentimento, quando aplicável.

5. Segurança
Adotamos medidas técnicas e administrativas aptas a proteger os dados pessoais contra acessos não autorizados, perdas, alterações ou divulgações indevidas.

6. Contato
Para dúvidas, solicitações ou exercício de seus direitos como titular de dados pessoais, entre em contato com nosso Encarregado pelo Tratamento de Dados Pessoais (DPO) pelo e-mail: basicprojeto@gmail.com
